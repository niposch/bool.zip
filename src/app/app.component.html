Description

This application can be used to minimize boolean functions. It uses the Quine-McCluskey Algorithm.

Symbols

* symbolizes the logical and operation.

+ symbolizes the logical or operation.

! symbolizes the negation.

- symbolizes dont care (can either be 0 or 1).
